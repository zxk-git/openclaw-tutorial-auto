# ⚡ 快速开始

> 5 分钟从零搭建 OpenClaw Agent 并运行第一个自动化任务

---

## 前置要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Linux / macOS / Windows (WSL2) |
| Node.js | 18+ |
| 网络 | 可访问 GitHub |

---

## Step 1：安装 OpenClaw

```bash
curl -fsSL https://openclaw.dev/install.sh | bash
```

验证安装：
```bash
openclaw --version
# 预期输出: openclaw 2026.x.x
```

---

## Step 2：初始化配置

```bash
openclaw init
```

这会创建 `~/.openclaw/` 目录并生成默认配置文件。

---

## Step 3：启动 Gateway

```bash
openclaw start
```

> [!TIP]
> 首次启动会自动配置设备身份和网络，可能需要 10-30 秒。

---

## Step 4：安装一个 Skill

```bash
# 从 ClawdHub 安装社区 Skill
openclaw skill install web-searcher
```

---

## Step 5：运行自动化任务

```bash
# 发送一条消息测试
openclaw message send --channel feishu --target <your-chat-id> --message "Hello from OpenClaw!"

# 创建定时任务
openclaw cron add --name "daily-report" --schedule "0 9 * * *" --command "openclaw run generate-report"
```
