# 📦 基础介绍与安装

> **难度**: ⭐ 入门 | **预计阅读**: 3 分钟

---

## 📋 本章要点

- OpenClaw 是开源多智能体自动化平台，LLM 驱动，可执行文件操作、API 调用、定时任务等
- 支持 Skills 插件扩展、飞书/Slack/Discord 多通道消息连接
- 内置长期记忆（Memory）与人格设定（SOUL）
- 系统最低要求：Node.js v20+、2GB 内存、500MB 磁盘
- 提供一键脚本、npm 全局、源码部署、Docker 四种安装方式
- 安装后通过 `openclaw onboard --install-daemon` 完成初始化
- 核心配置文件位于 `~/.openclaw/openclaw.json`
- 工作区目录 `~/.openclaw/workspace/` 存放身份、记忆、技能等文件

## 🔑 核心知识

OpenClaw 与传统自动化工具（Zapier、Cron 脚本）的最大区别在于其 AI 原生架构：LLM 驱动决策、内置 Agent 编排、社区共享的 Skills 插件体系以及长期记忆能力。这让它不仅能执行预设规则，还能理解复杂意图并自主完成多步骤任务。

安装完成后，核心目录结构包含 `workspace/`（工作区，含 SOUL.md、IDENTITY.md 等身份文件）、`skills/`（全局技能）、`cron/`（定时任务）、`credentials/`（凭证）和 `logs/`（日志）。`openclaw.json` 是全局配置入口，包含 AI 模型、代理、通道等所有关键设置。建议首次安装后运行 `openclaw doctor` 做一次完整的环境诊断。

## 💻 关键命令/代码

```bash
# 一键安装（推荐）
curl -fsSL https://openclaw.ai/install.sh | bash

# 初始化并安装守护进程
openclaw onboard --install-daemon

# 环境诊断
openclaw doctor
```
