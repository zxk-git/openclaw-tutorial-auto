# 第一章：OpenClaw 基础介绍与安装

> 本章适合初学者，从零开始了解 OpenClaw 基本概念和安装流程。

---

## 1.1 OpenClaw 简介

OpenClaw 是一套开源的多智能体自动化平台，具备 AI 执行、技能扩展和多通道连接能力。用户可通过指令、代码或技能扩展来定制自己的数字助手。

- **官网**：https://openclaw.ai
- **文档**：https://docs.openclaw.ai
- **GitHub**：https://github.com/openclaw/openclaw

### 核心特点
- 自动化任务执行与多智能体协作
- Skills 插件体系，功能可无限扩展
- 支持飞书、Slack、Discord 等多通道消息连接
- 支持本地部署与云端服务
- 开源、社区活跃

---

## 1.2 环境准备

### 系统要求
- **推荐平台**：Linux（如 Ubuntu 22.04+）、MacOS、Windows（需 WSL2）
- **所需组件**：Node.js v22+、npm、git

### 依赖安装示例（以 Ubuntu/Linux 为例）

```bash
sudo apt update
sudo apt install -y curl git

# 推荐使用 nvm 管理 node 版本
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
export NVM_DIR="$([ -z "${XDG_CONFIG_HOME-}" ] && printf %s "${HOME}/.nvm" || printf %s "$XDG_CONFIG_HOME/nvm")"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
nvm install 22
nvm use 22
```

---

## 1.3 OpenClaw 安装步骤

### 方式一：一键安装脚本（推荐）

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### 方式二：手动 npm 全局安装

```bash
npm install -g openclaw@latest
```

### 方式三：源代码部署（开发/贡献用）

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm run build
```

### Docker 部署（可选）

如偏好容器化部署，请见[官方 Docker 文档](https://docs.openclaw.ai/zh-CN/install/docker)。

---

## 1.4 初始化与基本配置

安装后首选初始化命令：

```bash
openclaw onboard --install-daemon
```

### 常见检查与启动命令

```bash
openclaw status        # 查看主服务状态
openclaw doctor        # 系统依赖和配置检测
openclaw dashboard     # 打开本地仪表盘（Web UI）
openclaw gateway start # 手动启动主服务
```

如因全局 npm 路径未加入 PATH，需额外设置：

```bash
export PATH="$(npm prefix -g)/bin:$PATH"
```

配置文件位于 `~/.openclaw/openclaw.json`，可自定义 AI Token、代理等参数。

## 1.5 目录结构简介

OpenClaw 安装后核心目录结构：

```
~/.openclaw/
├── workspace/                # 工作区目录
│   ├── AGENTS.md             # 多智能体协作模式
│   ├── SOUL.md               # 行为规范与人格设定
│   ├── TOOLS.md              # 工具配置
│   ├── MEMORY.md             # 长期记忆
│   └── memory/               # 日志记忆文件
│       └── YYYY-MM-DD.md
├── skills/                   # 已安装技能
│   └── <skill-name>/
│       └── SKILL.md
├── hooks/                    # 自定义钩子
│   └── <hook-name>/
│       └── HOOK.md
├── openclaw.json             # 主配置文件
└── ...
```

## 1.6 常见问题排查

| 问题 | 解决方法 |
|------|---------|
| 命令未找到 | 检查 PATH 是否包含 npm 全局 bin 目录 |
| Node 版本过低 | 用 nvm 安装 v22+ |
| Gateway 无法启动 | 运行 `openclaw doctor` 检测依赖与配置 |
| 权限问题 | 使用 sudo 或修复文件权限 |

## 本章小结

- OpenClaw 是开源多智能体平台，支持 AI 自动化与技能扩展。
- 安装推荐使用一键脚本或 npm 全局安装。
- 安装后通过 `openclaw onboard` 完成初始化。
- 常用命令：`status`、`doctor`、`dashboard`、`gateway start`。
- 配置文件位于 `~/.openclaw/openclaw.json`。
