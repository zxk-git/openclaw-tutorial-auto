# 第11章：高级场景：第三方平台集成

> 本章讲解 OpenClaw 与第三方平台的深度集成，包括 Google Workspace、Notion、GitHub 等。

---

## 11.1 Google Workspace 集成

通过 `gog` Skill 集成 Gmail、Calendar、Drive 等。

```bash
gog gmail search "keyword"     # 搜索邮件
gog gmail send to@email.com    # 发送邮件
gog calendar events            # 查看日程
gog drive search "filename"    # 搜索文件
gog docs export doc-id         # 导出文档
gog sheets get sheet-id        # 读取表格
```

---

## 11.2 GitHub 集成

通过 `github` Skill 使用 gh CLI。

```bash
gh pr list             # 列出 PR
gh issue list          # 列出 Issues
gh run list            # CI 运行列表
gh api repos/owner/repo # API 调用
```

---

## 11.3 Notion 集成

通过 `notion` Skill 管理 Notion 页面和数据库。

配置：
```json
{
  "env": {
    "NOTION_KEY": "your-notion-api-key"
  }
}
```
